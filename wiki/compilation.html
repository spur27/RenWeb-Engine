Building from Source

Build RenWeb for your platform using the included makefile. Supports 18 architectures across Linux, Windows, and macOS.

Linux

# Install dependencies
sudo apt install build-essential make pkg-config git libgtk-3-dev libwebkit2gtk-4.1-dev

# Optional: Install Boost libraries (if not using submodule)
sudo apt install libboost-all-dev

# Clone and build
git clone --recurse-submodules https://github.com/spur27/RenWeb-Engine.git
cd RenWeb-Engine
make TARGET=release
# Install dependencies
sudo dnf install gcc gcc-c++ make pkgconf-pkg-config git gtk3-devel webkit2gtk4.1-devel

# Optional: Install Boost libraries (if not using submodule)
sudo dnf install boost-devel

# Clone and build
git clone --recurse-submodules https://github.com/spur27/RenWeb-Engine.git
cd RenWeb-Engine
make TARGET=release
# Install dependencies
sudo pacman -S base-devel make pkg-config git gtk3 webkit2gtk

# Optional: Install Boost libraries (if not using submodule)
sudo pacman -S boost

# Clone and build
git clone --recurse-submodules https://github.com/spur27/RenWeb-Engine.git
cd RenWeb-Engine
make TARGET=release

Windows

Important: The makefile requires Git Bash or a Unix-compatible shell on Windows.

Important: MSVC's cl.exe compiler (from Visual Studio) is required - MinGW will not work.

# Install Visual Studio 2019+ with C++ workload and Windows SDK 10.0+

# Download WebView2 Runtime (Evergreen Standalone Installer)
curl -L -o MicrosoftEdgeWebview2Setup.exe "https://go.microsoft.com/fwlink/p/?LinkId=2124703"
# Run the installer: ./MicrosoftEdgeWebview2Setup.exe

# Optional: Download Boost installer (if not using submodule)
# https://sourceforge.net/projects/boost/files/boost-binaries/

# Clone repository (in Git Bash)
git clone --recurse-submodules https://github.com/spur27/RenWeb-Engine.git
cd RenWeb-Engine

# Build from Git Bash (using Developer Command Prompt environment)
make TARGET=release

macOS

# Install Xcode Command Line Tools
xcode-select --install

# Optional: Install Boost via Homebrew (if not using submodule)
brew install boost

# Clone and build
git clone --recurse-submodules https://github.com/spur27/RenWeb-Engine.git
cd RenWeb-Engine
make TARGET=release

Makefile Commands

The makefile supports various build targets and parameters for different architectures and configurations.

Build Commands

Command Description
make Build with default settings (debug mode)
make TARGET=debug Build in debug mode with debugging symbols
make TARGET=release Build in release mode with optimizations
make sub_modules Build the external submodules
make clean Clean up the build directory
make run Build and run the application
make test Build and test the application
make info Display build configuration information
make help Display help message with all commands

Cross-Compilation Parameters

Linux Cross-Compilation

Use the TOOLCHAIN parameter to cross-compile for different architectures:

# Example: Build for ARM64 on Linux
make TOOLCHAIN=aarch64-linux-gnu TARGET=release

# Example: Build for Raspberry Pi (ARM32)
make TOOLCHAIN=arm-linux-gnueabihf TARGET=release

Windows and macOS

Use the ARCH parameter to specify the target architecture:

# Windows: Build for ARM64
make ARCH=arm64 TARGET=release

# macOS: Build for x86_64
make ARCH=x86_64 TARGET=release

Multi-Architecture Builds

The build_all_archs.sh script automates building for all supported architectures:

# Build all architectures
./build_all_archs.sh

# Clone with submodules
git clone --recurse-submodules https://github.com/spur27/RenWeb-Engine.git

# Or initialize after cloning
