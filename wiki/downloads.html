CLI Tool

The renweb-cli package provides the rw command for scaffolding, building, running, and packaging RenWeb applications. Requires Node.js 16 or later.

Install globally via npm

npm install -g renweb-cli

Install from source

Clone the repository and link the CLI globally using the npm workspace script:

git clone https://github.com/spur27/RenWeb-Engine
cd RenWeb-Engine
npm run link:cli

See the CLI Tool documentation for all available commands.

JS / TS API

The RenWeb API package that allows you to interact with the RenWeb engine from your JavaScript or TypeScript code.

npm

npm install renweb-api

JSR (Deno / Bun)

# Deno
deno add jsr:@spur27/renweb-api

# Bun
bunx jsr add @spur27/renweb-api

Usage

const { System } = require('renweb-api');

const os = await System.getOS();
const arch = await System.getCPUArchitecture();
console.log(`OS: ${os}, Architecture: ${arch}`);

See the JS API documentation for the full API reference.
