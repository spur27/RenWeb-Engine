Bundles

Bundles include the RenWeb executable as well as all runtime libraries and dependencies. To use them, simply merge the bundle directory with a RenWeb directory to produce a directory with this structure.

Loading bundles...
NOTE: When running the program, make sure you launch it from the bundle-exec.sh script: 
chmod +x bundle-exec.sh
./bundle-exec.sh
Loading bundles...
NOTE: When running the program, make sure you launch it from the bundle-exec.bat script: 
bundle-exec.bat
MacOS has no dependencies and doesn't need bundles. You can run it with just the executable!
